Добавление клиента в книгу учета

Чтобы добавить клиента в книгу учета, необходимо в поля для заполнения внести номер абонемента, ФИО клиента, его телефон, выбрать тип абонемента(дневной, вечерний, круглосуточный) и добавить количество оплаченых занятий. После этого необходимо нажать на зеленую кнопку "Добавить запись".
